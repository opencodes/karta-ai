# 🎨 Karta: Visual Identity System

## 1. Color Palette
| Purpose | Color Name | Hex Code | Vibe |
| :--- | :--- | :--- | :--- |
| **Primary** | Midnight Navy | `#0A192F` | Trust, Depth, Finance |
| **Accent** | Electric Teal | `#64FFDA` | AI, Action, Energy |
| **Secondary**| Slate Grey | `#8892B0` | Modern, Professional |
| **Success** | Growth Green | `#4CAF50` | Positive Cashflow |

## 2. Typography
- **Headings:** *Inter* or *Montserrat* (Bold/Semi-bold) for a clean, tech feel.
- **Body:** *Roboto* or *Open Sans* for high readability in data tables.

## 3. Logo Concepts
- **The "K-Check":** A minimalist letter 'K' where the arm of the K is a checkmark.
- **The "Pulse-K":** A 'K' integrated with a subtle heartbeat/data pulse line.
- **Iconography:** Use "Linear" icons (thin lines) rather than solid blocks to keep the UI feeling "Light & Fast."
