# MVP_001: Karta-Do (Business Rules)

## Goal
Create tasks from one natural-language command with no manual form filling.

## Scope
- Convert command to structured task.
- Auto-set `title`, `category`, `due_date`.
- Save task for authenticated user.
- Show task in correct list bucket.

## Required Task Fields
- `raw_input`
- `title`
- `category` (`Finance`, `Personal`, `Work`, `Contact`, `General`)
- `due_date`
- `status` (default `pending`)
- `featured` (default `false`)

## Parsing Rules
- Remove conversational filler from title.
- Infer category from intent; fallback to `General`.
- Infer date/time from command.
- Apply default date/time policy if missing.

## AI Policy
- Use AI parser when available.
- If AI fails/unavailable, use deterministic fallback parser.
- Persist only valid parsed tasks.

## Bucketing Rules
- `now`: due in next 4 hours.
- `later`: due after 4 hours.
- `featured`: marked featured.
- `all`: all tasks.

## Validation
- Reject empty/invalid command.
- Do not store invalid parse output.

## Done Criteria
- One command creates one valid task.
- Task appears in correct bucket.
- Category is auto-assigned.
- Fallback works when AI is unavailable.
